Different Transformations

Transformations types

  • scale
  • skew
  • rotate
  • translate

Descriptions

  1. Scale :- Used to change the size of an element , along x and y axis
  2. Skew :- Used to distort an element along x and y axis
  3. Rotate :- Used to rotate an element around its center point
  4. Translate :- Used to move an element along x and y axis
