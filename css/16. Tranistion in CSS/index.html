What is Transition in CSS?

Transition in CSS is used to create smooth animations when changing CSS properties. It allows you to specify the duration, timing function, and delay of the transition effect.

  • Duration: Specifies how long the transition takes to complete.
  • Timing Function: Defines the speed curve of the transition.
  • Delay: Specifies a delay before the transition starts.
  • Property: Specifies which CSS property to apply the transition to.
