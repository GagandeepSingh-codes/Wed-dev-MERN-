HTML (Hper text mark language) is the standard markup language for creating web pages and web applications. It provides the basic structure of a webpage, which is then enhanced and modified by other technologies like CSS (Cascading Style Sheets) and JavaScript. No logical operations can be performed using HTML alone, as it is not a programming language but a markup language used to define the content and layout of a webpage.
